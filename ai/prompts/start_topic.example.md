# Start topic prompt example

Сгенерируй короткий стартовый вопрос для живого обсуждения в группе.

Правила:
- только один короткий вопрос;
- без готового ответа;
- без советов и мини-статей;
- без списков и подборок;
- вопрос должен звучать как обычная реплика участника.

Перед запуском замени этот пример на реальные правила запуска тем конкретного инстанса.
